# ИЭТР — Двухканальный блок питания на LM350

Интерактивное электронное техническое руководство. МИЭМ НИУ ВШЭ, Департамент компьютерной инженерии, 2026.

## Запуск

```bash
cd IETR
python3 -m http.server 8080
```

Открыть в браузере: [http://localhost:8080](http://localhost:8080)

> Запуск через локальный сервер обязателен: страницы подгружаются через `fetch()`, и при открытии `index.html` напрямую (file://) браузер заблокирует загрузку.

## Структура

| Файл | Назначение |
|------|-----------|
| `index.html` | Каркас: боковое меню и контейнер контента |
| `style.css` | Стили |
| `script.js` | Загрузка разделов (`loadPage`, `loadPageAndScroll`) |
| `pages/0.html` | Информативная справка (разработчики, руководство, источники) |
| `pages/1.html` | Общие сведения (описание, ТТХ, условия эксплуатации) |
| `pages/2.html` | Состав изделия (ведомость ЭРИ, конструктив, стандартизация) |
| `pages/3.html` | Техническая характеристика (надёжность, схема, топология, 3D, механика) |
| `pages/4.html` | Сборка и монтаж (видео, технологическая карта) |
| `pages/5.html` | Меры безопасности (частота, электробезопасность, IP, ЭМС, неисправности) |
| `pages/6.html` | Экономический расчёт |
| `images/` | Иллюстрации (извлечены из презентации НАША.pptx) |
| `video/` | Видео сборки (`assembly.mp4` — добавить) |

## Что нужно дополнить

- Поместить видео сборки в `video/assembly.mp4` и раскомментировать тег `<video>` в `pages/4.html`.
- При необходимости указать ФИО научного руководителя в `pages/0.html` (раздел «Академическое руководство»).
